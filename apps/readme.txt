i.MX6 Platform SDK Applications
===============================

The Platform SDK comes with a number of applications to demonstrate features of
both the SDK and underlying hardware.

caam_blob_gen       - Cryptographic blob generator for manufacturing encrypted boot images.
filesystem          - FAT32 filesystem performance test.
gpu_demo            - GPU demonstration.
httpd               - Web server.
multicore_demo      - Example showing how to start secondary CPU cores.
obds                - On-Board Diagnostic System used for manufacturing test of boards.
ping                - ICMP echo test.
power_modes_test    - Example of how to enter and exit low power modes.
sdk_unit_test       - Shell application for SDK driver unit tests.
stream              - Public domain CPU benchmark.
usb_hid_mouse       - Example USB device HID class application.
